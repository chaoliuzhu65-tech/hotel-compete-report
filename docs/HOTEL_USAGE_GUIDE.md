## 三、如何为新酒店生成报告

**步骤：**

1. **获取酒店坐标**
   - 打开高德地图APP → 搜索酒店 → 长按地图获取坐标
   - 或在高德开放平台 POI 搜索中获取

2. **确认竞品圆形（可选，系统自动完成）**
   - 系统默认5km圆形，如需调整（如城郊酒店可扩大至10km），告知AI即可

3. **告知AI执行**
   ```
   输入："竞品价格监控 [酒店名称] [日期] 坐标 [经度,纬度]"
   例："竞品价格监控 重庆名都酒店 2026-10-01至10-07 坐标 106.551,29.563"
   ```

4. **查看报告**
   - 系统生成HTML报告，自动推送至GitHub Pages
   - 可在飞书/钉钉群接收报告链接

## 四、常见问题

**Q：为什么我酒店附近没有找到竞品？**  
A：可能原因：①坐标偏移（确认经纬度精确度）；②半径内确实缺少同档位酒店（如度假村选址，建议扩大至15km）；③告知AI扩大搜索范围即可。

**Q：竞品价格与我在OTA看到的不一致？**  
A：飞猪数据为实时抓取，但受限于会员价/活动价差异，可能存在5-15%的偏差。建议用作**参考基准**而非精确值。

**Q：外资联号酒店（万豪/希尔顿）在报告里出现了，要跟他们对标吗？**  
A：**不要直接对标。** 外资联号品牌在报告中仅作为"市场上限参考"，德胧内资品牌有自己的价格锚点和客群，盲目追高外资价格会导致流失客源。

**Q：系统提示"假繁荣"，我该怎么选策略？**  
A：假繁荣=官方数据热闹但酒店端客质较弱（穷游/特种兵为主）。建议：①优先保守策略；②密切关注本店实际到店量；③如前三天实际入住率 >75%，可切换均衡策略。
