## 三、为新酒店生成竞品价格报告

### 3.1 准备输入JSON

创建一个新的JSON文件，比如 `my_hotel.json`：

```json
{
  "version": "3.1",
  "description": "XX酒店五一竞品价格分析",
  "hotels": [
    {
      "name": "XX开元名都大酒店",
      "brand": "开元名都",
      "star": "五星级/豪华型",
      "weekday_price": 458,
      "target_date": "2026-05-01",
      "lat": 30.3186,
      "lon": 120.2138,
      "district": "XX市XX区",
      "max_distance_km": 5,
      "scenario": "downtown"
    }
  ]
}
```

### 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| name | ✅ | 酒店全称 | `杭州开元名都大酒店` |
| brand | ✅ | 品牌 | `开元名都` |
| star | ✅ | 星级档次 | `五星级/豪华型` |
| weekday_price | ✅ | 平日基准价（元） | `458` |
| target_date | ✅ | 目标日期 | `2026-05-01` |
| lat | ✅ | 纬度 | `30.3186` |
| lon | ✅ | 经度 | `120.2138` |
| district | ✅ | 所在区域 | `杭州市萧山区` |
| max_distance_km | ⭕ | 搜索半径（km），默认5 | `5` |
| scenario | ⭕ | 场景预设：`downtown/resort/price_battle/default` | `downtown` |

**场景预设选择：**

| 场景 | 说明 | 权重配置 |
|------|------|----------|
| `downtown` | 市中心酒店 | 距离40% / 星级25% / 品牌15% / 价格20% |
| `resort` | 度假景区酒店 | 距离20% / 星级40% / 品牌25% / 价格15% |
| `price_battle` | 价格战区域 | 距离25% / 星级25% / 品牌10% / 价格40% |
| `default` | 默认 | 30%/30%/20%/20% |

### 3.2 经纬度获取方法

在高德地图搜索酒店，右键点击位置 → "这儿在哪里" 就能看到经纬度。

### 3.3 生成报告

```bash
python scripts/generate_batch_reports.py \
  --input my_hotel.json \
  --output output_reports/ \
  --api-key 0f9da10a87fa96c564f2d3d0f459fd6f
```

### 3.4 获取结果

生成完成后，在输出目录会得到：

```
output_reports/
├── README.md              # 报告索引
├── 酒店名称.md             # Markdown版本
└── 酒店名称.html           # HTML版本（手机友好，可直接打开）
```

## 五、每日自动监测（定时任务）

### 配置Cron定时任务

```bash
crontab -e
```

添加：

```cron
# 每天早上8点生成明天的竞品价格报告，推送到飞书
0 8 * * * cd /path/to/hotel-compete-report && \
  python scripts/build_compete_report.py \
    --hotel "天津瑞湾开元名都" \
    --target-date $(date -d+1day +\%Y-\%m-\%d) \
    --base-price 443 \
    --notify
```

## 七、常见问题

### Q: 提示"No module named 'amap'"怎么办？
A: 运行 `pip install amap-sdk --break-system-packages`

### Q: 筛选出来的有效竞品很少，正常吗？
A: 正常。严格过滤是为了保证可比性。如果周边真的没有同档次酒店，结果少反而更准确。

### Q: 价格数据都是真实的吗？
A: POI信息从高德API实时获取，价格数据从飞猪API实时获取，都是真实数据。测试阶段如果没有飞猪KEY，会使用估算涨幅，正式使用需要配置飞猪KEY。
